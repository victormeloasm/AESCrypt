Overview

AESCrypt Argon 6.0AK is the latest version of the AESCrypt tool, featuring AES-256-GCM encryption combined with Argon2id password hashing. This version provides unparalleled security, ensuring the protection of your sensitive data against modern cryptographic attacks. The tool also includes features such as secure file deletion, metadata protection, anti-keylogger measures, and much more. With its user-friendly interface, AESCrypt Argon 6.0AK is suitable for both everyday users and professionals who need top-tier data protection.

Features

  • AES-256-GCM Encryption: The most secure and trusted encryption standard available today.
  • Password Hashing with Argon2id: A powerful defense against brute-force attacks.
  • Secure File Deletion: Permanently erases files by overwriting them multiple times.
  • Metadata Encryption: Protects not only the contents but also the metadata of files.
  • Batch Encryption: Encrypt multiple files or entire directories in one action.
  • Cross-Platform Compatibility: Works seamlessly on both Windows and Linux systems.
  • Modern User Interface: A sleek dark mode interface for easy navigation.
  • Random Password Generation: Generate strong, random passwords for enhanced security.
  • Anti-Keylogger Protection: Integrated mechanisms to block keyloggers and protect your passwords.

Installation

Follow these steps to install AESCrypt Argon 6.0AK on your system:

Prerequisites

Ensure Python is installed on your system. You can download it from python.org.

Cloning the Repository

Clone the official AESCrypt repository from GitHub using:

git clone https://github.com/victormeloasm/AESCrypt.git

Installing Dependencies

Install the required dependencies with:

pip install -r requirements.txt

Compiling Executable (Optional)

For a standalone executable, use PyInstaller:

pyinstaller --onefile aescrypt.py

Virtual Environment (Optional)

Set up a virtual environment for dependency management:

python -m venv aescryptenv

Activate it with:

source aescryptenv/bin/activate  # Linux/macOS
aescryptenv\Scripts\activate  # Windows

Usage

Encrypt and decrypt files and folders with ease:

Encrypting Files

  1. Click "Browse File" to select the file.
  2. Enter or generate a secure password.
  3. Click "Encrypt" to save the encrypted file.

Decrypting Files

  1. Select the encrypted file.
  2. Enter the password used for encryption.
  3. Click "Decrypt" to restore the file.

Security Considerations

  • Use Strong Passwords: Always use a unique, strong password for encryption.
  • Memory Handling: Sensitive data is erased from memory when no longer needed.
  • Protection Against Keyloggers: Integrated measures block keyloggers during usage.
