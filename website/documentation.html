Overview

AESCrypt Argon 4.6A is a sophisticated file and folder encryption application designed for secure data protection. Utilizing advanced cryptographic methods, it implements Argon2 for password hashing and AES-256-GCM for encryption, ensuring the highest levels of security for sensitive information. This document provides a detailed exploration of the program's structure, functionalities, and underlying security mechanisms.

Features

  • AES-256-GCM Encryption: Provides confidentiality and data integrity with secure encryption and tamper detection.
  • Password Hashing with Argon2id: Utilizes Argon2id for robust password security, resistant to modern attacks.
  • Secure File Deletion: Implements secure deletion by overwriting original files before removal, preventing unauthorized recovery.
  • Metadata Encryption: Protects file-related information from exposure.
  • Batch Encryption: Encrypts all files within a folder with just a few clicks.
  • Cross-Platform Compatibility: Available for both Windows and Linux users.
  • User-Friendly GUI: Features a modern interface with a dark theme for enhanced user experience.
  • Automatic Password Generation: Offers a feature to generate strong, random passwords, enhancing security by preventing weak password choices.
  • Random Byte Addition: Adds random bytes to disguise file sizes, improving data privacy.
  • Memory Management Improvements: Cleans sensitive data from memory after use, reducing the risk of data leakage.
  • Improved Security Features: Increased iterations of Argon2id to enhance password hashing security.
  • Data Cleaning: Implements routines to clear sensitive data from memory after use, ensuring no residual data is left.

Installation

To install AESCrypt Argon 4.6A, follow these steps:

Prerequisites

Ensure you have Python installed on your system. Download it from python.org.

Cloning the Repository

Clone the AESCrypt repository using Git:

git clone https://github.com/victormeloasm/AESCrypt.git

Installing Dependencies

Navigate to the cloned directory and install the required packages:

pip install -r requirements.txt

Usage

To start using AESCrypt Argon 4.6A:

Launching the Application

You can run the application by executing the script in a terminal:

python AESCrypt.py

If compiled into an executable, double-click the generated `.exe` file.

User Interface Overview

The application interface is structured as follows:

  • File/Folder Path: Field for inputting the file or folder to encrypt or decrypt.
  • Password Entry: Field to enter a password, with an option to generate a random one.
  • Action Buttons: Buttons to initiate encryption, decryption, or password generation.

Encrypting Files

  1. Select a file by clicking "Browse File".
  2. Enter a secure password or click "Generate Password".
  3. Click "Encrypt" to secure the file. The output will have an `.aes` extension.

Decrypting Files

  1. Select the `.aes` file to decrypt.
  2. Enter the corresponding password.
  3. Click "Decrypt" to restore the original file.

Encrypting Folders

  1. Click "Browse Folder" to select a folder for encryption.
  2. Enter or generate a password.
  3. Click "Encrypt" to encrypt all files within the selected folder.

Security Considerations

Security is a critical component of AESCrypt Argon 4.6A. Key considerations include:

  • Strong Passwords: Users are encouraged to create strong and unique passwords for encryption to mitigate the risk of unauthorized access.
  • Secure Memory Management: The application is designed to clear sensitive data from memory post-use, reducing the risk of data leakage.
  • Regular Updates: Keep the software updated to ensure you have the latest security enhancements and bug fixes.
  • Data Backups: Regularly back up sensitive data to prevent loss or corruption, especially before performing encryption operations.
