Overview

AESCrypt Argon 5.0 is a highly sophisticated tool designed to provide advanced encryption and decryption of files and folders. This version leverages cutting-edge cryptographic algorithms such as AES-256-GCM for encryption and Argon2 for password hashing. Together, these technologies ensure robust data protection and resilience against modern attacks. The application also includes innovative features such as secure file deletion, metadata encryption, and anti-keylogger measures for added security. Whether you're a casual user or a security-conscious professional, AESCrypt provides an accessible and powerful solution for safeguarding your sensitive data.

Features

  • AES-256-GCM Encryption: The program uses AES-256-GCM, a symmetric encryption algorithm known for its high security and support for authenticated encryption, ensuring that data is both confidential and tamper-proof.
  • Password Hashing with Argon2id: This password hashing technique offers exceptional resistance against brute force and side-channel attacks, making it one of the most secure options available.
  • Secure File Deletion: When files are deleted, they are overwritten with random data multiple times, preventing their recovery by unauthorized parties.
  • Metadata Encryption: AESCrypt protects not only the content of the files but also metadata, such as file names and modification times, which are often overlooked during encryption processes.
  • Batch Encryption: Encrypt multiple files or an entire folder with a single action, improving both speed and efficiency.
  • Cross-Platform Compatibility: Available on both Windows and Linux, ensuring users on different platforms can benefit from its capabilities.
  • Enhanced User Interface: A modern, sleek GUI that provides an intuitive experience, especially with its dark theme that reduces eye strain during long sessions.
  • Random Password Generation: Generate strong, random passwords for encryption and decryption, ensuring that weak passwords are never an issue.
  • Random Byte Addition: Introduces random data into encrypted files to obscure their size and make it more difficult to identify encrypted files based on their file size.
  • Memory Management Improvements: The application ensures that sensitive data, such as passwords and encryption keys, are securely erased from memory after use, preventing potential data leakage.
  • Anti-Keylogger Protection: AESCrypt integrates advanced techniques to prevent keyloggers from capturing sensitive data such as passwords during encryption and decryption processes. By using secure input methods and memory isolation, the program reduces the risk of keylogging attacks, ensuring that user credentials remain safe from external threats.

Installation

Follow these steps to install AESCrypt Argon 5.0:

Prerequisites

Ensure Python is installed on your system. You can download it from python.org.

Cloning the Repository

Clone the AESCrypt repository from GitHub:

git clone https://github.com/victormeloasm/AESCrypt.git

Installing Dependencies

Navigate to the cloned directory and install the required Python dependencies:

pip install -r requirements.txt

Usage

To start using AESCrypt Argon 5.0, follow the instructions below:

Launching the Application

You can either run the script via a terminal or double-click the compiled `.exe` file if you have the executable version.

Encrypting Files

  1. Select a file by clicking the "Browse File" button.
  2. Enter a secure password or generate a new one by clicking "Generate Password".
  3. Click "Encrypt" to encrypt the file, which will be saved with the `.aes` extension.

Decrypting Files

  1. Click "Browse File" to select the `.aes` file you want to decrypt.
  2. Enter the password used during encryption.
  3. Click "Decrypt" to restore the original file.

Encrypting Folders

  1. Select a folder by clicking "Browse Folder".
  2. Enter a password for encryption.
  3. Click "Encrypt" to secure all files within the folder.

Security Considerations

Security is our top priority, and AESCrypt Argon 5.0 incorporates the latest security features to protect your data:

  • Strong Passwords: Always use a strong, unique password for encryption to ensure maximum protection against brute-force attacks.
  • Secure Memory Management: Sensitive data such as encryption keys and passwords are immediately erased from memory after use to prevent potential leaks.
  • Protection Against Keyloggers: With the integrated anti-keylogger protection, AESCrypt ensures that no keylogging software can intercept your passwords during use.
  • Frequent Updates: Keep AESCrypt updated to benefit from the latest security improvements and bug fixes.
  • Data Backup: Always back up your important files before encrypting them to avoid data loss in case of unforeseen issues.
