Overview

AESCrypt Argon 2025AK is an advanced encryption tool designed for individuals and organizations requiring high-level security. This version includes per-block encryption, optimized multithreading, secure deletion, and Anti-Keylogger protection.

The software encrypts files using AES-256-GCM and derives encryption keys using Argon2id, making brute-force attacks virtually impossible. With its optimized encryption engine, AESCrypt 2025AK provides efficient file protection while ensuring a seamless user experience.

Features

  • AES-256-GCM Encryption: Ensures data confidentiality, authenticity, and integrity.
  • Argon2id Password Hashing: Provides enhanced brute-force resistance.
  • Secure File Deletion: Overwrites files multiple times with random data to prevent recovery.
  • Random Byte Padding: Prevents attackers from inferring original file size.
  • Multithreading Support: Uses multiple CPU cores to accelerate encryption and decryption.
  • Anti-Keylogger Virtual Keyboard: Prevents keyloggers from capturing sensitive information.
  • Metadata Encryption: Protects filenames and file attributes.
  • Cross-Platform Compatibility: Available for Windows and Linux.

Installation

Prerequisites

Ensure Python is installed. Download from python.org.

Cloning the Repository

git clone https://github.com/victormeloasm/AESCrypt.git

Installing Dependencies

pip install -r requirements.txt

Compiling Executables

pyinstaller --onefile aescrypt.py

Usage

Encrypting Files

  1. Open AESCrypt 2025AK.
  2. Select a file using the "Browse" button.
  3. Enter a password or generate a random one.
  4. Click "Encrypt" to secure your file.

Decrypting Files

  1. Open the encrypted file.
  2. Enter the correct password.
  3. Click "Decrypt" to restore the original file.

Security Considerations

  • Use strong, unique passwords for encryption.
  • Enable checksum verification to ensure file integrity.
  • Keep backups of encrypted files to prevent accidental data loss.
  • Some antivirus software may flag AESCrypt due to its cryptographic operations. This is a **false positive**.
