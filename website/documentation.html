Overview

AESCrypt Argon 5.0 is a state-of-the-art encryption tool designed to provide secure, robust, and user-friendly encryption and decryption of files and folders. This version integrates AES-256-GCM encryption alongside Argon2id hashing for password protection, delivering a formidable defense against modern cryptographic attacks. The application offers secure file deletion, metadata protection, and an anti-keylogger mechanism. AESCrypt is built to be accessible to everyday users, while also ensuring professionals have the tools needed to safeguard their sensitive data efficiently and securely.

Features

  • AES-256-GCM Encryption: AES-256-GCM is a widely recognized encryption standard used for secure data encryption. The GCM mode provides additional integrity checks, making encrypted data both confidential and tamper-proof.
  • Password Hashing with Argon2id: Argon2id offers exceptional protection against brute-force and side-channel attacks, making it one of the most secure password hashing algorithms available.
  • Secure File Deletion: Ensures that files are securely deleted from the disk by overwriting them with random data multiple times, preventing file recovery by unauthorized parties.
  • Metadata Encryption: Protects not only the contents of your files but also their metadata, such as filenames, timestamps, and file sizes, which could otherwise leak information.
  • Batch Encryption: Encrypt multiple files or entire directories in one go, saving time and enhancing productivity when dealing with numerous sensitive files.
  • Cross-Platform Compatibility: Available for both Windows and Linux systems, ensuring that users across different platforms can benefit from the tool.
  • Modern User Interface: A sleek, dark-themed GUI that reduces eye strain while making it easy to navigate through the encryption and decryption process.
  • Random Password Generation: Generate strong, random passwords with a click, ensuring robust encryption without the risk of weak passwords.
  • Random Byte Injection: Introduces random data into encrypted files, obscuring the original file size and making it more difficult for attackers to identify encrypted files based on their size.
  • Memory Management Improvements: Sensitive data such as encryption keys and passwords are securely erased from memory once they're no longer needed, ensuring that they cannot be leaked.
  • Anti-Keylogger Protection: With integrated measures to block keyloggers, AESCrypt helps to prevent your passwords from being intercepted by malicious software during encryption and decryption.

Installation

Follow the steps below to install AESCrypt Argon 5.0 on your system:

Prerequisites

Before you install AESCrypt, ensure that Python is installed on your system. You can download the latest version from python.org.

Cloning the Repository

Clone the official AESCrypt repository from GitHub using the following command:

git clone https://github.com/victormeloasm/AESCrypt.git

Installing Dependencies

After cloning the repository, navigate to the project directory and install the required dependencies by running the command:

pip install -r requirements.txt

Compiling Executable (Optional)

If you wish to compile the tool into an executable file for easier use, you can use PyInstaller:

pyinstaller --onefile aescrypt.py

This command will generate a standalone `.exe` file that you can run directly on your system without needing to install Python.

Usage

Follow these steps to encrypt and decrypt files and folders using AESCrypt Argon 5.0:

Launching the Application

To start using the application, simply run the Python script via terminal or double-click the `.exe` file if you have the compiled version.

Encrypting Files

  1. Click "Browse File" to select the file you wish to encrypt.
  2. Enter a secure password or generate one using the "Generate Password" button.
  3. Click "Encrypt" to encrypt the file. It will be saved with the ".aes" extension.

Decrypting Files

  1. Select the encrypted file by clicking "Browse File".
  2. Enter the password used for encryption.
  3. Click "Decrypt" to restore the original file.

Encrypting Folders

  1. Click "Browse Folder" to select the folder containing the files you wish to encrypt.
  2. Enter a secure password for encryption.
  3. Click "Encrypt" to encrypt all files in the folder.

Security Considerations

Security is paramount when handling sensitive data, and AESCrypt Argon 5.0 ensures that your files and passwords are well-protected:

  • Use Strong Passwords: Always choose strong, unique passwords to enhance the encryption's effectiveness against brute-force attacks.
  • Secure Memory Handling: All sensitive information, such as passwords and encryption keys, are erased from memory as soon as they are no longer required, reducing the risk of data leakage.
  • Protection Against Keyloggers: AESCrypt incorporates anti-keylogger protection to safeguard your passwords from being intercepted by malicious software.
  • Keep Software Updated: Always ensure that you are using the latest version of AESCrypt to benefit from the newest security features and bug fixes.
  • Backup Your Files: Always create backups of important files before encrypting them, as data loss can occur due to unforeseen circumstances.
