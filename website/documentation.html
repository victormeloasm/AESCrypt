Overview

AESCrypt Argon 7.0AK REV represents a significant advancement in encryption technology, combining AES-256-GCM encryption with Argon2id password hashing and cutting-edge features like Anti-Keylogger protection. The REV edition introduces optimized block-based encryption, improved performance for large files, and enhanced user interface elements for a seamless experience.

The tool ensures the highest level of security for sensitive data while offering a user-friendly interface compatible with Windows and Linux. It supports large files, batch processing, and multithreading for faster and more efficient encryption and decryption tasks.

Features

  • AES-256-GCM Encryption: Ensures data confidentiality, authenticity, and integrity.
  • Argon2id Password Hashing: Protects passwords against brute force and side-channel attacks.
  • Secure File Deletion: Overwrites files with random data to prevent recovery.
  • Metadata Encryption: Encrypts metadata to prevent leakage of sensitive information.
  • Batch Processing: Efficiently encrypt or decrypt multiple files or folders simultaneously.
  • Large File Support: Optimized for handling files larger than 1GB with consistent performance.
  • Cross-Platform Compatibility: Available for both Windows and Linux.
  • Anti-Keylogger Protection: Prevents malicious software from capturing keystrokes during encryption or decryption.
  • Password Generator: Generates secure, random passwords for robust encryption.
  • Multithreading: Accelerates encryption and decryption using multi-core processors.
  • Random Byte Padding: Adds random data to prevent file size leakage.
  • Enhanced GUI: Features a modern, dark-themed interface for improved usability.

Installation

Follow these steps to install AESCrypt Argon 7.0AK REV:

Prerequisites

Ensure Python is installed. Download it from python.org.

Cloning the Repository

git clone https://github.com/victormeloasm/AESCrypt.git

Installing Dependencies

pip install -r requirements.txt

Compiling Executables

pyinstaller --onefile aescrypt.py

Usage

Encrypting Files

  1. Select a file using the "Browse File" button.
  2. Enter a password or generate one using the built-in generator.
  3. Click "Encrypt" to create an encrypted version of the file.

Decrypting Files

  1. Choose the encrypted file you want to decrypt.
  2. Enter the correct password.
  3. Click "Decrypt" to restore the original file.

Batch Encryption

Encrypt multiple files or an entire folder in one operation, saving time and effort.

Handling Large Files

AESCrypt Argon 7.0AK REV efficiently handles large files by processing them in blocks, ensuring consistent performance without memory issues.

Security Considerations

  • Use strong, unique passwords for each encryption task.
  • Verify the integrity of files using the built-in checksum feature.
  • Protect your system from malware to ensure secure operation.
  • Back up your encrypted files regularly to prevent data loss.
